# 1. AI Workflow总体设计

## 1.1 核心理念

系统不是：

用户输入问题 → AI回答。

而是：

用户提出目标 → AI团队完成任务。

---

用户目标：

> 我想做一条吸引健康用户咨询的视频。

AI自动完成：

```text
理解目标

↓

分析用户

↓

策划内容

↓

生成脚本

↓

优化审核

↓

制作视频

↓

生成运营方案
```

# 4. AI Agent设计

系统包含六个核心AI专家。

---

# Agent 1

# Content Expert（内容专家）

## 目标

解决：

主播不知道拍什么。

---

## 输入

来源：

### A

AI推荐

---

### B

爆款解析

---

### C

自定义主题

---

输入：

```json
{
"topic":"睡眠",
"user_id":1001
}
```

---

## 工作内容

分析：

1. 当前热点；
2. 用户画像；
3. 历史表现；
4. 咨询价值。

---

## 输出

```json
{
"topic":"45岁女性睡眠改善",
"audience":"中年女性",
"reason":"咨询潜力高",
"score":92
}
```

---

# Agent 2

# Planning Expert（策划专家）

## 目标

决定：

这条视频怎么讲。

---

输入：

Content结果。

---

输出：

```json
{
"purpose":"建立信任",
"emotion":"共鸣",
"style":"生活故事",
"scene":"厨房",
"duration":60
}
```

---

## 策划原则

优先：

故事。

经历。

生活场景。

不要：

产品介绍。

---

# Agent 3

# Script Expert（文案专家）

## 目标

生成主播可以直接拍的视频文案。

---

输出三个版本：

---

## Version A

故事型

适合：

40-55女性主播。

结构：

```text
开头冲突

↓

真实经历

↓

知识分享

↓

互动问题

↓

私信引导
```

---

## Version B

知识型

结构：

```text
问题

↓

原因

↓

方法

↓

行动
```

---

## Version C

聊天型

像朋友交流。

---

# 文案质量评分

AI自动评分：

| 指标     | 比例  |
| ------ | --- |
| 前三秒吸引力 | 25% |
| 共鸣     | 25% |
| 专业价值   | 20% |
| 咨询潜力   | 20% |
| 广告感控制  | 10% |

---

# Agent 4

# Compliance Expert（审核专家）

这是大健康行业非常重要的模块。

---

## 审核内容

### 医疗风险

禁止：

治疗。

治愈。

保证效果。

---

### 广告风险

降低：

买。

优惠。

立即购买。

---

### 内容价值

增加：

经验。

知识。

陪伴。

---

输出：

```json
{
"pass":true,
"risk":10,
"suggestions":[
"增加生活案例"
]
}
```

---

# Agent 5

# Video Expert（视频专家）

## 目标

把文案变成视频。

---

输入：

脚本。

---

输出：

视频任务。

---

流程：

```text
脚本

↓

声音

↓

真人素材匹配

↓

口型同步

↓

字幕

↓

BGM

↓

调色

↓

封面
```

---

# 真人素材模式（重点）

根据之前你的需求：

主播提前录制：

大量不同场景素材。

例如：

厨房。

散步。

喝水。

聊天。

运动。

家庭。

---

AI自动：

根据文案。

匹配素材。

---

例如：

文案：

“晚上睡不好”

匹配：

卧室。

喝茶。

思考。

---

# Agent 6

# Operation Expert（运营专家）

负责：

发布辅助。

---

输出：

标题：

3个。

标签：

10个。

评论：

3条。

私信引导：

1条。

---

例如：

标题：

《45岁以后，很多人的睡眠问题不是因为累》

---

评论：

“你有没有这种情况？”

---

私信：

“如果你也有类似困扰，可以交流一下。”

采用评分模型。

---

```

+
历史表现20%

+
咨询潜力20%
```

主题：

睡眠。

推荐。

# 7. 爆款解析Workflow

主播复制视频链接。

流程：

```text
视频链接

↓

解析视频

↓

提取字幕

↓

分析结构

↓

寻找爆点

↓

生成原创方案
```

---

输出：

不是复制。

而是：

```text
原视频:

为什么火？

↓

你的版本:

如何重新表达？
```
